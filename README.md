# Advandism-Avatar-Optimizer
It's a personal project in the direction of optimizing avatars by any means, having it robust and effective, while keeping it user friendly.
This project is going to be expanded, it will include more tools and features to optimize your own avatars with ease.

Currently supports:
- Triangle simplification heavily modified version of [Unity Mesh Simplification](https://github.com/Whinarn/UnityMeshSimplifier)
- Material merger (partial shader compability, work in progress)
- Mesh merger

## How to Use
In "Tools" you open the UI and see for yourself.

## Installation
Open your VRChat Creator Companion (VCC) and add to the custom repo this URL ´https://heftwig.github.io/Advandism-Avatar-Optimizer/index.json´
